About Pauper MTG Analytics

Created in late March 2025, stopped being updated in June 2025 because MTGO replays were broken and I restarted the project in April 2026 doin a huge overhaul.

Project is active again and data is up to date.
Almost everything can be clicked now. Give it a try!

Made with help from the AIs: Claude and Grok3 (original project) and then Codex for the overhaul

MTGO data gathered by Kirblinxy

Paupergeddon data by Lega Pauper Italia

Upstate NY data by Upstate NY Pauper (Bluesky)

Contact (for bugs or to send data): X/Twitter or Discord (ppavel)

Recent Improvements

Event Analysis and Player Analysis overhaul: Improved filtering (including the ability to filter the recent sets), few bug fixes and more ways to see the data. The dataset/archive workflow was also improved. Theme was changed and now we have light and dark modes.

Matchup Matrix (Head-to-Head) added, Elo system added. It is missing some minor functionalities from the quick cards (stat-cards), but the core functionality is there.

Known Issues

  • The dashboard still needs more work for smaller screens and full mobile optimization.
  • Some decks are classified as UNKNOWN because they could not be labeled cleanly from the source data. In MTGO this is either due to replays being unavailable because of a bug in MTGO or Kirb couldn't watch the replays for whatever other reason.
  • We do not have a real database pipeline yet, so things are still a little laggy, but the data refresh flow was improved.

Next Up in Priority Order

  • Add the player vs player matchup matrix and keep expanding matchup tooling.
  • Create ELO ratings for players. 1500 start point k=16, like in Vintage, with option to reset every season
  • Add Export as CSV functionality
  • Improve the mobile and small-screen layout.
  • Continue the move from one large frontend dataset file toward a cleaner JSON or database-backed pipeline.
  • Add more formats beyond Pauper.
  • Rework the mobile layout (do a full overhaul: Change statistics cards, less cluttered charts and tables, etc.)
